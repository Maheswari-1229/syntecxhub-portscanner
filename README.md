# Advanced TCP Port Scanner

## Overview

A multi-threaded TCP Port Scanner developed in Python for network reconnaissance and security assessment.

## Features

* TCP Port Scanning
* Multi-threaded Scanning
* Service Detection
* Banner Grabbing
* Logging to Files
* Exception Handling
* Scan Statistics
* Custom Port Range

## Technologies Used

* Python
* Socket Programming
* Threading

## Usage

```bash
python port_scanner.py
```

Enter:

* Target IP/Domain
* Start Port
* End Port

## Example

```text
Target: scanme.nmap.org
Start Port: 1
End Port: 1000
```
